Welcome to the CSC 4035 Lab 3 Website

Intro

This is a sample website created for CSC 4035 Lab 3. It includes a multi-page layout with a navigation bar, hero section, and multiple sections. The website is responsive and optimized for different screen sizes.

The purpose of creating the website is to learn how to:

  • Apply CSS3 for styling across multiple HTML pages.
  • Use flexbox and grid for layout.
  • Implement responsive design with media queries.
  • include animations and responsive media.
  • Maintain clean and reusable code structure across pages.
